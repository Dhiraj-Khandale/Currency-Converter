💱 Currency Converter
A simple and responsive Currency Converter web app built using HTML, CSS, and JavaScript that fetches real-time exchange rates using a live API.
🚀 Features

Convert between multiple world currencies instantly    
Real-time exchange rates fetched from a currency API    
Clean and user-friendly interface    
Responsive design for mobile and desktop    

🛠️ Tech Stack    
  HTML5 — Structure    
  CSs3 — Styling and layout    
  JavaScript (ES6+) — Logic and API integration    
  Exchange Rate API — Live currency data    

📂 Project Structure    
currency-converter/    
├── index.html    
├── style.css    
└── script.js    

⚙️ How to Use

Clone the repository:

bash   git clone https://github.com/Dhiraj-Khandale/Currency-Converter.git

Open index.html in your browser      
Select the currencies and enter an amount to convert
